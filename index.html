Títulos servem para definir o tema do site. existem seis níveis de títulos

Podemos destecar trechos de texto para enfatizar uma ideia!

  • Uma lista não ordenada não se importa com a ordem
  • UL significa "unordered list"
  • LI significa List Item
  1. Uma lista ordenada é uma lista que possue algum tipo de ordem númerica
  2. Seu funcionamento é exatamente como uma UL
  3. OL significa "ordered list"
